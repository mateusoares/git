Aprendendo Git e Html:

  • git clone: git clone acontece quando se copia tudo o que tem no repositorio github, para a maquina

  • git status: git status serve para saber as alteraçoes que o usuario fez

  • git add: git add serve para adicionar um arquivo ou uma alteraçao no repositorio
    ex: git add git.html ou git add . ou git add /pasta/.

  • git commit git commit serve para "confirmar" o que o usuario fez
    ex: git commit -m "texto dizendo o que voce commitou"

  • git push git push serve para enviar as alteraçoes feitas na maquina para o repositorio

  • git diff: git diff serve para ver o que foi alterado
    ex: git diff git.html
